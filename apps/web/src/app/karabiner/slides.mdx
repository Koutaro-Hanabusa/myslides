import { HeadingSlide, ContentSlide } from "./components/slide";

<HeadingSlide>
  Karabiner-Elementsで<br/>キーボードを自由自在に
</HeadingSlide>

<ContentSlide title="Karabiner-Elementsとは？">
  - macOS用のキーボードカスタマイズツール
  - 無料・オープンソース
  - キーのリマップ、複雑な入力ルールが設定可能
</ContentSlide>

<HeadingSlide>
  こんな悩みありませんか？
</HeadingSlide>

<ContentSlide>
  - CapsLockキー、一生使わない問題
  - JIS配列とUS配列を行き来する人生
  - Vimのescが遠すぎる問題
</ContentSlide>

<ContentSlide title="基本的な使い方">
  - **Simple Modifications**: 単純なキー置換
  - **Complex Modifications**: 条件付きの複雑なルール
  - **Devices**: デバイスごとの設定
</ContentSlide>

<HeadingSlide>
  おすすめ設定例
</HeadingSlide>

<ContentSlide>
  - `CapsLock` → `Control` (定番)
  - `CapsLock` → `esc` (Vimmer向け)
  - `英数/かな` → `Command` (US配列ユーザー向け)
</ContentSlide>

<ContentSlide title="Complex Modificationsの例">
  - `cmd + hjkl` で矢印キー
  - 左右Shiftでカッコ入力 `()` `{}`
  - アプリごとにキー設定を変更
</ContentSlide>

<HeadingSlide>
  デモ
</HeadingSlide>

<ContentSlide title="まとめ">
  - Karabinerでキーボードは自分仕様にできる
  - 生産性が爆上がり
  - 設定ファイルはJSONで管理・共有可能
</ContentSlide>
