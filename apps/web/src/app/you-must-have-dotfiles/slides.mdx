<HeadingSlide>こういう設定ファイルも<br />dotfilesで管理できると便利じゃない？</HeadingSlide>

<ContentSlide title="そもそもdotfilesとは">
  - zshやbashなどのホームディレクトリにある<br />ドットから始まるファイル(.zshrcなど)を管理しているリポジトリ
  - 複数のPC間で設定を共有することができる
</ContentSlide>

<ContentSlide title="dotfilesを管理することのメリット">
- 仕事用、私用PCなど異なるPC間で設定を共有できる
- 自分の設定の履歴を管理することができる
- 他の人が簡単に設定を参照できる(<del>そしてパクれる</del>)
</ContentSlide>

<HeadingSlide>よし、dotfilesで管理しましょう！！！！！</HeadingSlide>

<HeadingSlide>まずはchezmoiを使って気軽に管理し始めよう</HeadingSlide>

<ContentSlide title="chezmoiとは">
- [chezmoi(読みはシェモア)](https://github.com/twpayne/chezmoi)
- dotfilesを複数のPC間で安全に管理するためのgo製ツール
</ContentSlide>

<ContentSlide title="始め方">
※前提：mac ユーザーでclaude codeを使っている人

chezmoiをインストールする

```bash
brew install chezmoi
```

出力例
```
1126buri@MacBook-Air-2 ~ % brew install chezmoi
==> Fetching downloads for: chezmoi
✔︎ Bottle chezmoi (2.69.3) Downloaded 12.1MB
==> Pouring chezmoi--2.69.3.arm64_sequoia.bottle.tar.gz
🍺  /opt/homebrew/Cellar/chezmoi/2.69.3: 9 files, 33.3MB
```

</ContentSlide>
<ContentSlide>
<div className="h-full">

chezmoiを有効化する

```bash
chezmoi init
```

CLAUDE.mdや、skillsを管理下にする（mcp.jsonはAPIキーなどが含まれるのでやめましょう）

```bash
chezmoi add ~/.claude/CLAUDE.md ~/.claude/settings.json ~/.claude/commands ~/.claude/skills ~/.claude/agents ~/.claude/plugins/installed_plugins.json
```

管理下になったことを確認する

```bash
chezmoi managed
```

出力例(一部抜粋)
```
.claude/CLAUDE.md
.claude/agents/frontend-developer.md
.claude/plugins/installed_plugins.json
.claude/settings.json
.claude/skills/frontend-design/SKILL.md
```
</div>
</ContentSlide>

<ContentSlide>
chezmoiが管理しているディレクトリが生成されるので移動する
```bash
chezmoi cd
```
ホームディレクトリ配下にこのようなディレクトリが作成されていることを確認できればOK
```bash
pwd
# ~/.local/share/chezmoi
```
あとはいつも通りコミットしてPushすればOK!
</ContentSlide>


<ContentSlide title="まとめ">

- 他の人に設定を共有することも簡単になる(<del>パクりやすくなる</del>)
